# 随机市场的交易策略

这是一个针对 btc-updown-5m 或 btc-updown-15m 之类系列的交易策略。

它假设：
- 底层的资产（比如比特币）在较短时间尺度上是随机波动的，可以使用一个相对简单的数学公式来计算到结束时最终结果是上涨还是下降的概率分别是多少。
- PolyMarket 中包含大量非理性的人类交易者，人类交易者会受到情绪影响（比如因短时间剧烈波动而恐惧）导致市场价格偏离实际概率。如果价格偏离概率较多则存在套利空间。

因此基本逻辑是通过数学公式计算上涨和下降的概率，将概率与市场实际价格比较，如果相差大于指定阈值，则进行一次交易。

## 执行细节

- 交易至少间隔 30s ，如果 30s 内交易过，就跳过
- 市场结束前 30s 内不进行任何交易
- 交易操作包括 4 种，每次选择其中一种或不进行任何操作
  - 以市价单 FOK 类型买入 1 USD 当前市场 Yes(Up)
  - 以市价单 FOK 类型买入 1 USD 当前市场 No(Down)
  - 以市价单 FOK 类型卖出当前市场全部 Yes(Up) 持仓
  - 以市价单 FOK 类型卖出当前市场全部 No(Down) 持仓
  其中买入 Yes 和卖出 No 基本是等价的，买入 No 和卖出 Yes 基本是等价的，因此如果有对应持仓应该优先做卖出操作
- 所有 FOK 滑点保护设置为当时最佳买价、卖价 +-0.2 。比如 Sell 单的滑点保护价设置为最佳买价-0.2 。
- 参考 *数学原理* 小节的方式计算 Yes 概率 P(Yes) ，与当前 Yes 成交价格 Price(Yes) 进行比较，如果 P(Yes) - Price(Yes) > 0.2 表示当前 Yes 被显著低估（或者说 No 被高估），则应买入 Yes 或卖出 No 。如果 P(Yes) - Price(Yes) < -0.2 则表示 Yes 被显著高估，则应卖出 Yes 或买入 No 。

## 数学原理

考虑这样一个问题：假如有一个指标，每秒能随机向上或向下按一个固定幅度波动，在给定的时间后（比如 2 分钟），它能大于或者小于某个给定值的概率如何计算？

假设指标每秒的变动是独立的：每次有 50% 概率向上波动一个固定幅度 \(\sigma\)，50% 概率向下波动相同幅度 \(\sigma\)。经过 \(n\) 秒（例如 2 分钟 = 120 秒）后，指标值的分布与初始值 \(S_0\) 有关。根据波动率定义的不同，常见有两种模型：

1. **算术随机游走**（绝对波动）：\(S_{t} = S_{t-1} \pm \sigma\)
2. **几何随机游走**（相对波动）：\(S_{t} = S_{t-1} \cdot e^{\pm\sigma}\)（或 \(1 \pm \sigma\)）

以下分别给出精确与近似的概率计算方法。

### 算术随机游走（绝对波动）

设初始值为 \(S_0\)，每秒独立地加 \(\sigma\) 或减 \(\sigma\)，概率各半。  
\(n\) 秒后，价格变化量为 \(S_n - S_0 = (U - D)\sigma\)，其中 \(U\) 为向上次数，\(D\) 为向下次数，\(U+D = n\)。

于是 \(S_n = S_0 + (2U - n)\sigma\)，\(U \sim \text{Binomial}(n, 0.5)\)。

#### 精确概率
- 大于给定值 \(K\) 的概率：
  \[
  P(S_n > K) = P\!\left(U > \frac{n + \frac{K - S_0}{\sigma}}{2}\right)
  = \sum_{u = u_{\min}}^{n} \binom{n}{u} 0.5^n
  \]
  其中 \(u_{\min} = \left\lfloor \frac{n + \frac{K - S_0}{\sigma}}{2} \right\rfloor + 1\)（若该值小于0则取0，大于 \(n\) 则概率为0）。

- 小于给定值 \(K\) 的概率：
  \[
  P(S_n < K) = \sum_{u=0}^{u_{\max}} \binom{n}{u} 0.5^n
  \]
  其中 \(u_{\max} = \left\lceil \frac{n + \frac{K - S_0}{\sigma}}{2} \right\rceil - 1\)。

#### 正态近似（\(n\) 较大时）
当 \(n\) 较大（如 \(n=120\)）时，\(U\) 近似服从正态分布 \(\mathcal{N}(n/2, n/4)\)，从而 \(S_n\) 近似服从：
\[
S_n \sim \mathcal{N}\big(S_0,\; n\sigma^2\big)
\]
因此：
\[
P(S_n > K) \approx 1 - \Phi\!\left( \frac{K - S_0}{\sigma\sqrt{n}} \right)
\]
\[
P(S_n < K) \approx \Phi\!\left( \frac{K - S_0}{\sigma\sqrt{n}} \right)
\]
其中 \(\Phi\) 为标准正态累积分布函数。

#### 示例（2 分钟，120 秒）

假设初始值 \(S_0 = 100\)，\(\sigma = 0.5\)（绝对波动），\(K = 105\)。

- 精确计算：需用二项分布，\(n=120\)，可借助统计软件或二项分布表。
- 正态近似：\(\sigma\sqrt{n} = 0.5 \times \sqrt{120} \approx 5.477\)，  
  \(P(S_{120} > 105) \approx 1 - \Phi(5 / 5.477) = 1 - \Phi(0.913) \approx 0.180\)。

#### 关键说明

- **步长 \(\sigma\) 的确定**：若题目中“固定波动率”是年化标准差，需按时间缩放（如 \(\sigma_{\text{秒}} = \sigma_{\text{年}} \times \sqrt{1/31536000}\)），此时每秒步长仍为常数，上述方法适用。
- **非对称概率**：若向上/向下概率不是各半，只需调整二项分布的成功概率 \(p\)，正态近似时均值为 \(S_0 + n(2p-1)\sigma\)，方差为 \(4np(1-p)\sigma^2\)。
